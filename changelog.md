- **Five additive hardening improvements on the Smart IFEO Compatibility Layer (Detector A install-time AppX/browser drop + Detector B runtime SYSTEM-crash auto-exclude store), all backward-compatible (old 4-field store lines still parse; production build byte-for-byte unaffected by the test seam).** This hardens the layer that fixed Win11 Notepad (CLEAN-GUI refusal as SYSTEM) + Chrome/Firefox (crash/blank under SYSTEM):
